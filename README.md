# La Nave

Site institucional, link da bio e painel editorial conceitual para a La Nave Restaurante e Pizzaria.

## Stack

- React
- Vite
- Tailwind CSS
- Lucide React

## Rotas

- `/` site institucional
- `/bio` link da bio
- `/admin` painel editorial

## Scripts

```bash
npm install
npm run dev
npm run build
```
